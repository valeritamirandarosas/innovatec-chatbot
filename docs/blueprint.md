# **App Name**: Innovatec Chatbot

## Core Features:

- Autenticación: Autenticación segura de usuarios usando Google y Correo Electrónico/Contraseña.
- Chatbot de IA: Chatbot interactivo impulsado por Gemini 1.5 Flash, que proporciona asistencia educativa basada en el personaje y modo seleccionados, con indicaciones dinámicas.
- Planes de Aprendizaje Personalizados: Herramienta de IA para generar planes de aprendizaje, cuestionarios y tarjetas didácticas personalizados según la materia, el grado y el personaje seleccionado.
- Gestión de Perfiles de Usuario: Gestionar perfiles de usuario, incluyendo alias, progreso de estudio y estadísticas de gamificación, almacenados en Firestore.
- Gamificación y Tabla de Posiciones: Seguimiento del progreso del usuario con rachas, minutos estudiados, cuestionarios aprobados y puntajes, y mostrar una tabla de posiciones de los mejores usuarios.
- Recopilación de Comentarios: Recopilar comentarios de los usuarios a través de un formulario, almacenarlos en Firestore y, opcionalmente, enviar notificaciones por correo electrónico.
- Texto a Voz: Habilitar la síntesis de voz web para los mensajes del chatbot según la voz seleccionada.

## Style Guidelines:

- Color primario: Cian neón (#00E5FF) para reflejar la marca innovadora y enérgica de Innovatec. Este color aporta brillo a la interfaz de usuario.
- Color secundario: Violeta (#7C3AED) para complementar el color primario, creando una sensación moderna y vibrante. Este color sugiere inteligencia y conocimiento.
- Color de fondo: Azul marino oscuro (#0F172A) para un entorno de aprendizaje elegante y enfocado.
- Color de texto base: Gris claro (#E2E8F0) para garantizar la legibilidad contra el fondo oscuro.
- Fuente del cuerpo y del encabezado: 'Inter', una sans-serif para una legibilidad clara y una estética moderna, recurriendo a las fuentes de la interfaz de usuario del sistema para una compatibilidad más amplia.
- Utilizar iconos limpios y minimalistas para representar diferentes materias, modos y personajes.
- Un diseño limpio y responsivo optimizado tanto para escritorio como para dispositivos móviles, centrándose en la legibilidad y la facilidad de navegación. El uso destacado de los colores seleccionados dirigirá al usuario a la información importante. Cada personaje tiene un cuadro de texto único.
- Animaciones sutiles para comentarios, actualizaciones de progreso y transiciones de la interfaz de usuario para mejorar la experiencia del usuario sin ser molestas.
